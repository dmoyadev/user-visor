## Project setup
```
npm install
```

Add a `.env` file in the root directory with two properties:
```
VUE_APP_API_URL = 'https://randomuser.me/'
VUE_APP_GOOGLE_API_KEY = <custom key of Google Maps API>
```

### Compile and minify for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
